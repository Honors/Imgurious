Imgurious

Imgurious is an imgur client for the web utilizing aggressive server-side caching for efficient, scalable use as a proxy of the site's assets.

Dependencies

  • Java >= 1.6
  • Conscript
  • giter8

Build & Run

Run the Scala Build Tool with the following.

$ ./sbt.sh

Then, from within its interactive shell, run the following.

$ ./sbt.sh
> ~;container:start; container:reload /
